## 🌟 Sobre o Projeto

O **Semeando Sorrisos** é uma iniciativa social independente que atua em Goiânia, realizando ações voluntárias diretas. Mais do que apenas entregar doações, o foco do projeto é a presença, o cuidado e a construção de vínculos com aqueles que mais precisam.

Esta landing page foi desenvolvida para centralizar as informações do projeto, facilitar a captação de voluntários, organizar a arrecadação de doações materiais e dar transparência ao impacto gerado na comunidade.

## 🚀 Funcionalidades

- **Design Mobile-First:** Experiência totalmente otimizada para celulares, garantindo acessibilidade em qualquer dispositivo.
- **Seção de Impacto:** Painel visual com indicadores de famílias apoiadas e toneladas de alimentos arrecadados.
- **Gestão de Doações:** Listagem clara de itens prioritários (Roupas, Alimentos, Higiene e Brinquedos).
- **Formulário de Voluntariado:** Canal direto para novos membros se inscreverem.
- **Localização em Tempo Real:** Integração com Google Maps para facilitar a entrega de doações no ponto de coleta.
- **FAQ Interativo:** Seção de dúvidas frequentes utilizando componentes do tipo *accordion*.

## 🛠 Tecnologias

O projeto foi construído utilizando as melhores práticas de desenvolvimento web moderno:

- **React.js**: Biblioteca principal para construção da interface.
- **CSS Modules / Styled Components**: Para estilização fluida e componentizada.
- **Vite**: Ferramenta de build rápida e moderna.
- **Lucide React / Phosphor Icons**: Para iconografia suave e moderna.
- **Google Maps API**: Para integração de localização.

## 🔧 Como Executar o Projeto

```bash
# Clone o repositório
git clone [https://github.com/seu-usuario/semeando-sorrisos.git](https://github.com/seu-usuario/semeando-sorrisos.git)

# Entre na pasta do projeto
cd semeando-sorrisos

# Instale as dependências
npm install

# Inicie a aplicação em modo de desenvolvimento
npm run dev
```
